########################
Conocimientos esenciales
########################

Esta nueva versión de CROSS permite una mayor interacción de la organización con el cliente,
mejorando los procesos de control de tiempos en seguimientos y requerimientos; realiza
información crítica de cada cliente; controla los principales procesos involucrados en la 
relación Organización - Cliente.
