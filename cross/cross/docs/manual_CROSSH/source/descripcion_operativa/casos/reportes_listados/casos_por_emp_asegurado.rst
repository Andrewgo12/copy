#########################
Casos por Emp. Asegurado
########################

.. |info| image:: ../../../img/informacion.png
.. |advertencia| image:: ../../../img/alerta.png
.. |fecha| image:: ../../../img/fecha.png
.. |servicio| image:: ../../../img/servicio.png

Este reporte imforma la cantidad de casos por cada tipo de caso, clasificación o subclasificación impuestos por cada empresa.

+---------------+------------------------------------------------------------------------+
||advertencia|  | **Nota:**  Los campos acompañados por un asterisco ( * ) son de        | 
|               |                                                                        |
|               |  carácter obligatorio.                                                 |
+---------------+------------------------------------------------------------------------+

+---------------+------------------------------------------------------------------------+
| |info|        | **Nota:**  Por defecto la aplicación mostrará las tareas de su         | 
|               |                                                                        |
|               |   dependencia o ente organizacional.                                   |
+---------------+------------------------------------------------------------------------+
1. Ingrese a la opción **"Casos > Reporte y listados > Casos por Emp. Asegurado"** del árbol de 
   opciones que se encuentra a la izquierda de la pantalla. Esta acción mostrará la 
   siguiente pantalla:

   .. image:: ../../../img/casos_emp_asegurados.png
    :alt: Casos por Emp. Asegurado


+--------------------+---------------------------------------------------------------------+
|Campo 	             | Descripción                                                         |
+====================+=====================================================================+
| Registro           | Seleccione usando el botón |fecha| la fecha de registro.            |
|                    |                                                                     |
+--------------------+---------------------------------------------------------------------+
| Emp. aseguradora de| Seleccione de la lista existente, la aseguradora de salud, la cuál  |
| salud              |  desea consultar.                                                   |
+--------------------+---------------------------------------------------------------------+
| Tipo de caso       | Seleccione de la lista existente el tipo de caso el cuál            |
|                    | desea consultar.                                                    |
+--------------------+---------------------------------------------------------------------+
| Clasificaciones    | Seleccione el checkbox para incluir clasificaciones en la busqueda. |
|                    |                                                                     |
+--------------------+---------------------------------------------------------------------+
| Subclasificaciones | Seleccione el checkbox para incluir subclasificaciones en la        |
|                    | busqueda.                                                           |
+--------------------+---------------------------------------------------------------------+

2. Digite o seleccione la información requerida.

3. Para terminar presione el botón "Consultar" o pulse el botón "Limpiar" para cancelar esta acción y empezar de nuevo.
