## FLUJO DE EJECUCIÓN

```
1. ENTRADA
   index.php
   ↓
   
2. CONFIGURACIÓN
   config/config.inc.php
   ↓ Carga Application class
   ↓ Define directorios
   
3. FRONT CONTROLLER
   Web/FrontController.class.php
   ↓ WebRegistry::getWebCommand()
   ↓ Obtiene acción de $_REQUEST["action"]
   
4. COMANDO
   web/commands/FeCrCmd*.class.php
   ↓ execute()
   ↓ Lógica de negocio
   
5. MANAGER (si necesario)
   domain/FeCr*Manager.class.php
   ↓ Procesa datos
   
6. BASE DE DATOS (si necesario)
   data/Pgsql/FeCrPgsql*.class.php
   ↓ Operaciones CRUD
   
7. VISTA
   WebRegistry::getWebView()
   ↓ Carga template
   
8. TEMPLATE
   web/templates/Form_*.tpl
   ↓ TemplateView::show()
   ↓ Smarty rendering
   
9. SALIDA
   HTML generado al navegador
```

## MÓDULOS PRINCIPALES

### GESTIÓN DE ÓRDENES
- Crear, editar, eliminar órdenes
- Listado y consulta de órdenes
- Seguimiento de órdenes

### GESTIÓN DE SOLUCIONES
- Registro de soluciones
- Consulta de soluciones
- Anexos y documentos

### REPORTES
- Centro de reportes
- Reportes consolidados
- Reportes detallados
- Tiempos de ejecución

### ADMINISTRACIÓN
- Gestión de usuarios
- Administración de tareas
- Configuración del sistema
- Transferencia de tareas

### SOPORTE MULTIIDIOMA
- Español (es/)
- Inglés (en/)
- Archivos de traducción por módulo
